# Shops & Markets

TownifyMC economy is player-driven. Buying and selling happens through TinyMarkets player shops, AxPlayerWarps, and the AuctionHouse.

## Player markets

TinyMarkets lets players create physical shops using containers. These are the main way to buy and sell regular goods.

```
/market
```

Player shops are useful for passive income in towns, marketplace areas, or other high-traffic locations. Stock items players actually want, keep prices competitive, and advertise your location.

## Setting up your own shop

To set up a useful shop, you generally need:

1. **A place players can reach** — a town plot, marketplace spot, or other public build.
2. **Stock** — items players actually want.
3. **A clear price** — watch other shops and the auction house so your prices make sense.
4. **Traffic** — use a player warp once you unlock and create one.

## Player warps

AxPlayerWarps lets players create public warp points for shops and attractions.

```
/pwarp
/pwarps
/playerwarps
```

Creating a player warp costs <span class="chip money">$10,000</span>. Warp names are limited to 16 characters, cannot use spaces, and teleporting has a 3-second delay.

## The auction house

AuctionHouse supports both buy-it-now listings and bid auctions. Use it for rare items, high-value gear, limited event items, or anything you want the whole server to see.

```
/ah
/ah sell <price>
/ah bid <amount>
```

| Auction setting | Value |
|-----------------|------:|
| Tax | 0% |
| Default max auctions | 10 |
| Buy-it-now duration | 48 hours |
| Bid auction duration | 2 hours |
| Minimum listing price | $1 |

## Which should I use?

| Situation | Use |
|-----------|-----|
| Selling regular stocked goods | **Player shop / market** |
| Selling rare or high-value items | **Auction house** (`/ah`) |
| Building passive income | **Your own shop** + a **player warp** |
| Looking for deals | Browse **player markets** and **auctions** |
