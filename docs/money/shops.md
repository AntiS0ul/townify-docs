# Shops & Markets

TownifyMC has several ways to buy and sell: the server shop, TinyMarkets player shops, AxPlayerWarps, and the AuctionHouse.

## The server shop

```
/shop
```

Use the server shop for fixed-price buying and selling.

## Player markets

TinyMarkets lets players create physical shops using containers.

```
/market
```

Player shops support Vault currency trading and are useful for passive income in towns or marketplace areas.

## Player warps

AxPlayerWarps lets players create public warp points for shops and attractions.

```
/pwarp
/pwarps
/playerwarps
```

Creating a player warp costs <span class="chip money">$10,000</span>. Warp names are limited to 16 characters, cannot use spaces, and teleporting has a 3-second delay.

## The auction house

AuctionHouse supports both buy-it-now listings and bid auctions.

```
/ah
/ah sell <price>
/ah bid <amount>
```

| Auction setting | Value |
|-----------------|------:|
| Tax | 0% |
| Default max auctions | 10 |
| Buy-it-now duration | 48 hours |
| Bid auction duration | 2 hours |
| Minimum listing price | $1 |
