# Fishing

TownifyMC uses **EvenMoreFish** for custom fish, bait, sell menus, and competitions.

## Custom fish & rarities

Fishing can catch custom fish with rarity, size, and sell value. Catches are not limited to competitions, so you can fish for custom rewards any time.

```
/evenmorefish
/emf help
/emf shop
/emf sellall
/emf journal
```

## Rods & bait

EvenMoreFish has bait support. Bait boosts fish rarity odds, can be applied to rods, and each rod can hold up to **7** bait types. Bait is disabled during competitions to keep leaderboards fair.

## Fishing competitions

Competitions can resume after restarts and are held back if another competition is already active.

```
/emf top
/emf next
/emf toggle
```

## Why fish?

| Reason | Payoff |
|--------|--------|
| **Fisher job** | Earn job money on catches |
| **Fishing skill** | Level AuraSkills Fishing |
| **Selling catches** | Sell fish through `/emf shop` or `/emf sellall` |
| **Competitions** | Compete for timed leaderboard rewards |
