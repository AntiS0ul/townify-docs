# Skills

TownifyMC uses **AuraSkills** for RPG-style character progression. Skills level up automatically as you play and unlock passive bonuses, abilities, and mana abilities.

## How skills work

```
/skills
/stats
```

Each enabled skill has a maximum level of **100**.

## The skill set

=== "Gathering"

    | Skill | Mana ability |
    |-------|--------------|
    | **Farming** | Replenish |
    | **Foraging** | Treecapitator |
    | **Mining** | Speed Mine |
    | **Fishing** | Sharp Hook |
    | **Excavation** | Terraform |

=== "Combat"

    | Skill | Mana ability |
    |-------|--------------|
    | **Archery** | Charged Shot |
    | **Fighting** | Lightning Blade |
    | **Defense** | Absorption |

=== "Utility"

    | Skill | Notes |
    |-------|-------|
    | **Agility** | Movement and fall-related bonuses |
    | **Alchemy** | Brewing and potion-related bonuses |
    | **Enchanting** | Enchanting/anvil bonuses |

## Skills vs. Jobs

| | [Jobs](jobs.md) | Skills |
|---|---|---|
| **Reward** | Money | Stats, abilities, passive bonuses |
| **How you join** | Manually, limited slots | Automatic, all enabled skills |
| **What they are for** | Income and ranking up | Making your character stronger |
