# Quests & Battle Pass

TownifyMC has daily quests, daily/weekly/monthly login rewards, and a 30-day Battle Pass season.

## Daily Quests

Daily quests are powered by ODailyQuests.

```
/quests
```

Quests are grouped by difficulty and include objectives for normal play, such as mining, farming, fighting, and fishing.

## Login Rewards

TownifyMC has daily, weekly, and monthly login rewards.

```
/rewards
/dailyrewards:rewards
```

| Reward | Cooldown | Normal reward | Premium reward |
|--------|----------|---------------|----------------|
| Daily | 24 hours | $500 + 250 XP | $1,000 + 500 XP + 1 Vote key |
| Weekly | 168 hours | $2,500 + 1,250 XP + 1 Vote key | $5,000 + 2,500 XP + 2 Vote keys + 1 Epic key |
| Monthly | 720 hours | $7,500 + 5,000 XP + 2 Vote keys | $15,000 + 10,000 XP + 4 Vote keys + 1 Deluxe key |

## Battle Pass

TownifyMC runs a Battle Pass with free and premium tracks. Seasons last **30 days**, and battle coins reset when the season ends.

```
/battlepass
/bp
```

- Complete missions to earn Battle Pass XP.
- Claim rewards from the free track as you progress.
- Premium adds extra rewards on the premium track.
- The Battle Pass shop lets you spend Battle Pass currency.
