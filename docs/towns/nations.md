# Nations

Once your town is established, the next step is joining or founding a **nation** — a union of towns under a single banner.

## Unlocking nations

Creating a nation unlocks at **Warrior (rank #10)**, which costs <span class="chip money">$500,000</span> in rank progression. Founding the nation itself costs <span class="chip money">$10,000</span> from the town bank.

!!! tip
    You do not need to found a nation to be part of one. Any town can be invited into an existing nation.

## Founding a nation

```
/nation new <name>
```

### Core nation commands

```
/nation
/nation add <town>
/nation set capital <town>
/nation set spawn
/nation deposit <amount>
/nation ally add <nation>
/nation enemy add <nation>
```

## Why join a nation

| Benefit | What it means |
|---------|---------------|
| **Strength in numbers** | Allied towns can support each other in war |
| **Diplomacy** | Form alliances, declare enemies, negotiate |
| **Shared identity** | A nation tag and shared capital spawn |
| **Fast travel** | Travel through town and nation spawn networks |

## Spawn travel and neutrality

| Setting | Value |
|---------|------:|
| Town spawn travel | <span class="chip money">$0</span> |
| Nation neutrality upkeep | <span class="chip money">$100</span> |
| Town neutrality upkeep | <span class="chip money">$25</span> |
| Nation daily upkeep | <span class="chip money">$100</span> |

New nations are neutral by default in the Towny config, while new towns are not.
