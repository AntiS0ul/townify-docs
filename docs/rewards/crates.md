# Crates

TownifyMC uses ExcellentCrates. Open crates with matching keys for money, Gems, items, backpacks, spawners, and other rewards.

## How crates work

1. **Get a key** from voting, rewards, events, the store, or other systems.
2. **Go to the crates area** with `/warp crates`.
3. **Open the crate** or preview its rewards.

```
/crates
/key
/warp crates
```

## Getting more keys

| Source | Which keys |
|--------|-----------|
| [Daily voting](voting.md) | Vote Crate keys |
| Login rewards | Vote, Epic, and Deluxe keys depending on reward tier |
| Events and competitions | Varies |
| [Server store](https://store.townifymc.xyz/) | Premium keys |
| Battle Pass | Keys as tier rewards |
